# Implementation of an electronic calculator using plain javascript

## 1. Run using npm

### Install

```
npm install
```

### Run

```
npm start
```

### Access the application

The application is accessible at <b>[http://localhost](http://localhost)</b>

<hr/>

## 2. Run using Docker

```
docker-compose up
```

### Access the application

The application is accessible at <b>[http://localhost](http://localhost)</b>
